# Personal Portfolio Website

A clean, modern, and responsive personal portfolio website built using
HTML, CSS, and JavaScript.

## Features
- Professional UI
- Responsive design
- Colorful skill badges
- Smooth scrolling
- Easy to customize

## Technologies
- HTML5
- CSS3
- JavaScript

## How to Use
1. Download or clone this repository
2. Add your photo in images/profile.jpg
3. Update your details in index.html
4. Open index.html or deploy using Netlify / GitHub Pages
